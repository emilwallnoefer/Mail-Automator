# Post-mail asset descriptions - Français (FR)

> Modifier uniquement le texte sous `DESC:` pour chaque asset.
> Laisser les lignes `ID:`, `LABEL:` et `LINK:` inchangées pour que les
> modifications puissent être reportées dans `web/src/lib/change-options.ts`.
>
> Chaque asset apparaît comme une carte-lien dans l'e-mail récapitulatif :
>
> ```
> ➡️ [LABEL](URL)
> DESC
> ```
