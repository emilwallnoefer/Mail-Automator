# Post-mail asset descriptions - Deutsch (DE)

> Nur den Text unter `DESC:` für jedes Asset bearbeiten.
> Die Zeilen `ID:`, `LABEL:` und `LINK:` unverändert lassen, damit die
> Änderungen zurück nach `web/src/lib/change-options.ts` übernommen werden können.
>
> Jedes Asset erscheint als eine Link-Karte in der Trainings-Recap-E-Mail:
>
> ```
> ➡️ [LABEL](URL)
> DESC
> ```

---
